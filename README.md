# Mod Features


### Items & Blocks


Helmet, Chestplate, Leggings, Boots, Horse armor, Sword, Axe, Pickaxe, Shovel and Hoe


Tungsten Upgrade Template, Tungsten Block, Tungsten Ore


### Tungsten Food


Tungsten Slop Grants short duration effects: Regeneration, Resistance and Fire Resistance But Negative Hunger.

Tungsten Jerky Grants short duration effects: Strength and Mining Fatigue.


### Heavy Enchantment


Heavy is an enchantment compatible with armors, increasing an armors knockback resistance by 2.5% each level and also Applies an additional slowness penalty per level


### Advancements
Adds **Tungsten Advancement Page**: Powder?, Yummy Slop, Metallic Munchies, Netherite’s Equal, Rich in Tungsten, Hidden Treasure, Forged in Tungsten, Cover Me in Tungsten, Tooled Up in Tungsten and Tungsten Steed

# Progression & World Generation


### Progression
Mine **Tungsten Ore** → Smelt **Tungsten Powder** → Craft **Tungsten Ingots** → Upgrade Diamond gear using **Tungsten Upgrade Templates**  


### World Generation


**Upgrade Templates** spawn in Strongholds, Ancient Cities, Desert Temples, Jungle Temples, and dungeons


**Tungsten Scrap** has a small chance to spawn in Pillager Outposts, Buried Treasure Chests, Shipwreak Treasure Chests and Both Underwater Ruins


**Tungsten Ore** spawns in the **deepslate layer**, in scattered veins. Generation style similar to Ancient Debris
